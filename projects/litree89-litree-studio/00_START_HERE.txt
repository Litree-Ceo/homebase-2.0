❓ NEED HELP?
═══════════════════════════════════════════════════════════════════════════════

Issue: "Cannot find script"
  Fix: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Issue: "Port already in use"
  Fix: taskkill /PID <PID> /F
  Or: cd app && npm run dev -- --port 3000

Issue: "node_modules missing"
  Fix: .\litree.ps1 dev-start

Issue: "Build fails"
  Fix: npm run clean && npm run install:all && npm run build
