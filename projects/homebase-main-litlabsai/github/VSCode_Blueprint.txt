## 4. How to Control Everything
- Use the auto-git-sync.ps1 script for all git operations.
- Use the Syncing extension to keep settings/extensions synced across VS Code installs.
- All workspace settings are in .vscode/settings.json—edit this file for tweaks.
- Theme and icons are set for Honeycomb Vision style.
- All performance tweaks are applied for fast writing and terminal speed.
