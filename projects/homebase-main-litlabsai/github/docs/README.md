## 🎯 Quick Navigation

**New to HomeBase?** Start here:

1. Read [Getting Started](./getting-started/)
2. Set up your development environment
3. Check [Development](./development/) for contribution guidelines

**Deploying to production?**

1. Review [Deployment](./deployment/) guides
2. Follow the appropriate platform guide (Azure or Google Cloud)
3. Check [Operations](./operations/) for monitoring setup

**Something broken?**

1. Check [Operations](./operations/) troubleshooting guide
2. Review relevant [Reference](./reference/) documentation
3. Look in [Archives](./archives/) for historical context

## 📝 Contributing to Documentation

When adding new documentation:

1. Determine which section it belongs to
2. Add it to the appropriate subdirectory
3. Update the section's README to reference the new document
4. Ensure markdown is properly formatted
5. Use consistent header hierarchy
