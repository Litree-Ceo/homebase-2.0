## 🎯 How to Read the Dashboard

### Metric Meanings:

| Metric            | Meaning                | What's Good                       |
| ----------------- | ---------------------- | --------------------------------- |
| **ROI**           | Return on Investment % | > 10% monthly is excellent        |
| **Win Rate**      | % of profitable trades | > 60% is good, > 70% is great     |
| **Profit Factor** | Avg Win ÷ Avg Loss     | > 2.0 is good, > 3.0 is excellent |
| **Sharpe Ratio**  | Risk-adjusted returns  | > 1.0 is good, > 2.0 is excellent |
| **Max Drawdown**  | Worst losing period    | < 5% is good, < 2% is excellent   |

### Action Indicators:

| Signal                      | Action                                   |
| --------------------------- | ---------------------------------------- |
| 🟢 **Green** (> 10% ROI)    | Keep running, may increase position size |
| 🟡 **Yellow** (5-10% ROI)   | Running fine, monitor for improvements   |
| 🔴 **Red** (< 5% ROI)       | Review strategy, adjust parameters       |
| ⚫ **Black** (Negative ROI) | Pause bot, investigate, modify settings  |
