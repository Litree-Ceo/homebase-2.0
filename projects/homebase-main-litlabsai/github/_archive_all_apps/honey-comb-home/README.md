# Honey Comb Home

This is the main landing page and home application for the LitreeLabs platform.

## Purpose
- Serves as the primary landing page
- Provides authentication and user onboarding
- Acts as the entry point to other applications

## Technology Stack
- Static site served with npx serve
- Basic HTML/CSS/JS structure
- No build process required

## Development
```bash
npm run dev
npm run build
npm run start
```

## Deployment
Deploy using Azure Static Web Apps workflow or Azure CLI
