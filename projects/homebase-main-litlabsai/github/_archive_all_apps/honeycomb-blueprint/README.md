# Honeycomb Blueprint

This is a template/starter application for new projects in the LitreeLabs ecosystem.

## Purpose
- Serves as a blueprint/template for new applications
- Contains the basic structure and configuration
- Used as a starting point for new development

## Technology Stack
- Static site served with npx serve
- Basic HTML/CSS/JS structure
- No build process required

## Development
```bash
npm run dev
npm run build
npm run start
```

## Deployment
Deploy using Azure Static Web Apps workflow or Azure CLI
