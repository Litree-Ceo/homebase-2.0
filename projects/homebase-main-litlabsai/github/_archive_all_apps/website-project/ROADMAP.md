# LiTbiT-2.0 Roadmap

## Phases

1. **Planning & Research**
   - Market analysis, competitor review, user surveys
2. **MVP Build**
   - Wallet connect, basic social feed, one bot type
   - Smart contract and backend scaffolding
3. **Beta Testing**
   - Recruit testers, gather feedback, iterate
4. **Full Launch**
   - NFT drops, marketing, community events

## Milestones

- Smart contract deployment (Polygon/Base)
- Bot integration (AI, trading, NFT)
- Cosmos DB backend setup
- Security audits and compliance
- Community and marketing launch
