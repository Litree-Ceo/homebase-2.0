## Animation Showcase

```
Fade In:       Opacity goes from 0 to 1
               ░░░░░░░░░░ → ██████████

Slide In:      Element slides up from below
               (Position 10px below) → (Position 0)

Scale In:      Element scales from 95% to 100%
               [Smaller] → [Normal Size]

Pulse Soft:    Gentle opacity pulsing
               ██████████ ↔ ░░░░░░░░░░ (loop)

Transitions:   All properties smoothly animate
               Duration: 150-500ms
               Easing: Smooth cubic-bezier
```
