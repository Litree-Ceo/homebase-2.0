# Deployment Status Panel

This panel will show the current status of your Azure and Firebase deployments, including:

- Last deployment time
- Current branch deployed
- Build status (success/fail)
- Live links to logs and dashboards

## How to Use

- Integrate with your CI/CD workflows to update this file automatically.
- Use Copilot Chat or Azure extensions to fetch and display live deployment data.
