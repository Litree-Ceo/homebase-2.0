# LiTreeLab Studio Pad - Labs AI

This is the AI, content, and automation engine for the LiTreeLab Studio Pad platform.

## Features

- AI content generation (text, image, video, music)
- Multi-channel publishing (TikTok, Instagram, YouTube, etc.)
- Bot builder and automation
- Analytics and reporting
- Stripe payments and subscriptions

## Quick Start

1. Copy `.env.example` to `.env.local` and fill in your keys.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
