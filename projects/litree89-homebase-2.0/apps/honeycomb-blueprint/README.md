# LitreeLabs - Honeycomb Vision Workspace

## Honeycomb Vision Theme & Layout Quickstart

- **Themes**: All theme presets and tokens are in `config/themes.ts`. Use or extend as needed.
- **Switch themes**: Use `components/theme/ThemeSwitcher.tsx` in your dashboard or layout to let users pick a theme.
- **Dashboard layout**: The honeycomb hex grid is implemented in `dashboard/honeycomb/index.html` and styled by `dashboard/honeycomb/style.css`.
- **Preview**: Open `dashboard/honeycomb/index.html` in your browser for a live demo of the Honeycomb Vision layout.
