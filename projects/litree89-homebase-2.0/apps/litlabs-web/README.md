# LiTreeLab Studio Pad - Web App

This is the main web application for the LiTreeLab Studio Pad social/creator platform.

## Features

- Social feed, chat, notifications, media sharing
- AI content generation and automation
- Multi-provider authentication (Firebase, Azure, etc.)
- Stripe payments and subscriptions
- Admin dashboard and smart notifications

## Quick Start

1. Copy `.env.example` to `.env.local` and fill in your keys.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
