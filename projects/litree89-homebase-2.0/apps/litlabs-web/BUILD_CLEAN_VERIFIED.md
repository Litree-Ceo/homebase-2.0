## 📞 TROUBLESHOOTING

### Problem: `npm install` fails

**Solution**:

- Make sure Node.js v18+ is installed (`node -v`)
- Run: `npm cache clean --force`
- Then: `npm install` again

### Problem: `npm run dev` fails

**Solution**:

- Check: `.env.local` has Firebase keys
- Run: `npm install` again
- Check: No syntax errors in `types/*.ts` files

### Problem: Can't find component examples

**Solution**:

- Read: `JUMP_IN_GUIDE.md` (has LoginForm template)
- Review: `lib/*.ts` (shows usage patterns)
- Check: `types/*.ts` (shows data types)

### Problem: TypeScript errors

**Solution**:

- Check: `types/*.ts` files for type definitions
- Review: `config/*.ts` for configuration options
- Look for: JSDoc comments explaining usage
