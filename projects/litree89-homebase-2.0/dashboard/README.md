# EverythingHomebase Dashboard

This dashboard gives you quick control over your workspace settings, automation scripts, and future Azure visual integrations.

## Quick Change Guide

- **VSCode_Blueprint.txt**: Full setup blueprint for VS Code.
- **settings.json**: Workspace settings for performance, theme, and sync.
- **auto-git-sync.ps1**: One-command Git automation.
- **Nitro5_Master_Check.ps1**: System verification script.
- **azure-visuals/**: Placeholder for Azure dashboards and visuals.

## How to Use

- Edit any file here to instantly change your environment.
- Use the Syncing extension to keep everything synced across devices.
- Add Azure visuals and integrations in the `azure-visuals` folder.
