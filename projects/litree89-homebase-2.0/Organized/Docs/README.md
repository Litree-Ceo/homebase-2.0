# EverythingHomebase

Your all-in-one project homebase for code, docs, assets, and scripts.

## Quickstart

1. Clone the repo:

   ```sh
   git clone https://github.com/LiTree89/EverythingHomebase.git
   cd EverythingHomebase
   ```

2. Install dependencies:
   - Node.js: `npm install`
   - Python: `pip install -r requirements.txt`

3. Copy `.env.example` to `.env` and fill in secrets (never commit secrets).

4. Start the app:
   - Node.js: `npm start`
   - Python: `python src/main.py`

## Structure

- `src/` — Source code (Node.js, Python, etc.)
- `docs/` — Documentation
- `assets/` — Images, media, and other assets
- `scripts/` — Automation and utility scripts

## Usage

- See `docs/` for setup and usage instructions.
- Use `.env` for secrets (never commit this file).

## Security

**Important:**

- Never commit secrets or `.env` files to version control.
- Always use environment variables for sensitive data.
- Review cloud permissions and access regularly.
