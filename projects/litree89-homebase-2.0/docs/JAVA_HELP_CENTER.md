# Java Help Center in VS Code

Welcome to the Java Help Center. This guide provides quick access to resources and commands for Java development in VS Code.

## 📰 Stay Updated
- **Check Blogs**: View the latest updates and features via [Java in VS Code](https://code.visualstudio.com/docs/languages/java).

## 🚀 Get Started
- **Create New Project**: 
  - Press `Ctrl+Shift+P`
  - Type **Java: Create Java Project**
  - Select your build tool (Maven, Gradle, etc.) and location.
- **Open Existing Project**: 
  - Simply open a folder containing Java files (`File > Open Folder...`). VS Code will automatically activate Java support.
- **Take a Tour**: 
  - Run the command **Java: Tips for Beginners** to get an interactive walkthrough.

## 🛠️ General Features
- **IntelliSense**: Smart code completions, parameter info, and quick info.
- **Debugging**: Full support for breakpoints, variable inspection, and call stack.
- **Testing**: Integrated support for **JUnit** and **TestNG**.
- **Refactoring**: Move, rename, extract methods, and more.

## 🍃 Spring Boot
- **Microservices**: Use the **Spring Boot Extension Pack** for specialized tools.
- **Initializr**: Create new Spring Boot projects directly from VS Code.
- **Dashboard**: Manage and run your Spring Boot applications easily.
- **Getting Started**: Follow [Getting Started with Spring Boot in VS Code](./SPRING_BOOT_GETTING_STARTED.md) for the same extension + Initializr workflow showcased in the VS Code welcome card.

## 🎓 Student & Beginner Resources
- **Beginner Tips**: Access tailored guidance via the **Java: Tips for Beginners** command.

## ⚙️ Configuration
- **Java Runtime**: 
  - Ensure you have a JDK (1.8 or higher) installed.
  - Configure the JDK per project in your workspace settings.
- **Open Java Settings**: 
  - Press `Ctrl+Shift+P`
  - Type **Preferences: Open Settings**
  - Search for "java" to see all available configurations.
- **Formatter Settings**: 
  - Customize code style via `java.format.*` settings.

## 📦 Extensions
To get the best experience, ensure you have the following installed:
- **Extension Pack for Java** (Language support, Debugger, Maven, Gradle)
- **Spring Boot Extension Pack** (Spring support)
