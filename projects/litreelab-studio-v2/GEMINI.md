# LiTreeLabStudio Project Mandates

This file contains foundational mandates for Gemini CLI when working within this workspace.

## 🛠️ Tech Stack Preferences
- **Frontend**: React (TypeScript) with Vanilla CSS.
- **AI**: MCP-first architecture and Python/FastAPI backends.
- **Styling**: Modern, high-aesthetic UI/UX with a focus on "audiophile-grade" polish.

## 📜 Standards
- All new features MUST include documentation in the Workflows/ or Rules/ directories.
- Prioritize modularity and cross-project compatibility.
