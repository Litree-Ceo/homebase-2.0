## 🔧 NEXT STEPS - REQUIRED ACTIONS

### 1. Configure Your API Keys
Edit `.env` file and fill in your actual keys:

```bash
# Generate secure API key
python -c "import secrets; print(secrets.token_urlsafe(32))"

# Generate auth token
python -c "import secrets; print(secrets.token_hex(32))"
```

### 2. For Real-Debrid Streaming
Get your API key from: https://real-debrid.com/apitoken
Add to `.env`: `RD_API_KEY=your_key_here`

### 3. For Firebase (Optional)
1. Go to Firebase Console → Project Settings
2. Generate service account JSON
3. Add to `.env`: `FIREBASE_CONFIG={...}`

### 4. Run the Dashboard
```bash
# Start the Python server
python server.py

# Or use the launcher
powershell -ExecutionPolicy Bypass -File overlord-master.ps1
```
