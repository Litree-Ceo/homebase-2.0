Welcome to the **LiTreeLabStudio** Root Workspace. This is the centralized hub for all laboratory projects, AI research, and studio development.




## 📂 Workspace Structure
The **litlab-os-pad** is located at:
- `litlab's-player-deluxe/litrelab/litreelab-studio`


1. Navigate to the directory:
	- `cd litlab's-player-deluxe/litrelab/litreelab-studio`
2. Install dependencies:
	- `pnpm install`
3. Start the dev server:
	- `pnpm dev`
